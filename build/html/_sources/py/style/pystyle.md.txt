# Python Style

[Python PEP-8编码风格指南中文版](https://alvin.red/2017/10/07/python-pep-8/)

来自非英语国家的 Python 程序员们，请使用英文来写注释，除非你 120% 确定你的代码永远不会被不懂你所用语言的人阅读到。

## 英文写作指南

使用英文写作，参考 Strunk 和 White 的[《The Elements of Style》](http://www.huili.info/2011/01/15/%E3%80%8A%E8%8B%B1%E6%96%87%E5%86%99%E4%BD%9C%E6%8C%87%E5%8D%97%E3%80%8B%E3%80%8Athe-elements-of-style%E3%80%8B%E7%AC%AC%E4%B8%89%E7%89%88%E9%98%85%E8%AF%BB%E7%AC%94%E8%AE%B01/)
